# ICC T20 World Cup Dashboard

## Project Description

The ICC T20 World Cup Dashboard is a frontend web project designed to present important information about the T20 World Cup tournament in a structured and visually clean interface. The project demonstrates how HTML and CSS can be used to build a multi-page sports dashboard.

The website allows users to explore teams, match schedules, points tables, and player statistics. The main objective of this project is to practice frontend layout design and organize sports data in an accessible format.

---

## Features

* Multi-page website structure
* Dedicated pages for teams, schedule, points table and statistics
* Clean and structured layout
* Simple navigation system
* Organized tournament information display
* Beginner-friendly frontend code

---

## Technologies Used

* HTML5
* CSS3
* Flexbox / CSS Layout

## Pages Included

### Home Page

Displays tournament banner and highlights.

### Teams Page

Contains the list of all participating teams.

### Match Schedule

Shows all tournament fixtures with dates.

### Points Table

Displays team standings including matches played, wins, losses and points.

### Player Statistics

Shows top run scorers and top wicket takers.

## How to Run the Project

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in any web browser.
